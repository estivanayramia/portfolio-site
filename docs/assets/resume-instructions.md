## Format Tips
1. Keep it to 1-2 pages (1 page preferred for students)
2. Use a clean, professional template
3. Consistent formatting (fonts, spacing, alignment)
4. PDF format (not Word doc) - preserves formatting
5. Proofread carefully - no typos
6. Use bullet points, not paragraphs
7. Quantify achievements with numbers/percentages
8. Reverse chronological order (most recent first)
